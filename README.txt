How to compile this program (lab2):
For Mac/Linux:
	In terminal type:
	1.	cd (directory in which the files are contained);  
	2.	touch BMI.h;
			(update last modified time of BMI.h)
	3.	make;
	4.	./lab2;				
			(create an output file "file.out")
	5.	cat file.out
			(file contains BMI value and its corresponding category)
